# 首版开发计划

## Sprint 0：工程落地

目标：把基础工程、数据库和窗口骨架搭起来。

- Electron + React + Vite 工程初始化
- SQLite 初始化
- Preload API 设计
- 三栏布局占位
- 基础主题与导航

## Sprint 1：文献库 MVP

目标：实现“导入 -> 存储 -> 展示”。

- 选择文件导入
- 文件复制到应用数据目录
- 生成 `papers` 记录
- 文献列表页
- 文献详情页基础信息

## Sprint 2：解析与搜索

目标：让导入的数据可检索、可阅读。

- TXT/PDF 内容提取
- FTS 建索引
- 关键字搜索
- 按文件夹/标签筛选

## Sprint 3：笔记与分类

目标：支持持续沉淀。

- 标签管理
- 文件夹管理
- 笔记树
- 笔记与文献关联

## Sprint 4：AI 能力

目标：跑通“基于选中文献提问”。

- AI 配置页
- 预设模型切换
- 新建对话
- 选中文献作为上下文
- 流式回复
- 对话持久化

## Sprint 5：增强体验

目标：让产品更接近可用版本。

- AI 摘要回写文献
- 对话导出 Markdown
- 批量导入
- 重复文件检测
- 空状态和异常处理

## 推荐顺序

1. Sprint 0
2. Sprint 1
3. Sprint 2
4. Sprint 3
5. Sprint 4
6. Sprint 5
